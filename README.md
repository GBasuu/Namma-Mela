# Namma-Mela 🎭

## Android Theatre Seat Booking & Management App

Namma-Mela is a modern Android theatre booking application developed using Kotlin and Room Database for rural drama theatres and village melas.

## Features
- User Login & Registration
- Role-based Manager/Admin Access
- 5x5 Seat Booking System
- Room Database Persistence
- Multi-show Booking
- House Full Detection
- Waiting List System
- Fan Wall & Reviews
- Cast Management
- Manager Dashboard
- Modern Dark UI

## Technologies Used
- Kotlin
- XML
- Android Studio
- Room Database
- SQLite
- SharedPreferences
- Material Design

## Project Structure
- Activities
- Room Database
- DAO
- XML Layouts
- Authentication Module
- Manager Module

## How to Run
1. Clone repository
2. Open in Android Studio
3. Sync Gradle
4. Run app
